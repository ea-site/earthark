対応機種・PCブラウザについて

当フォーラムは従来の掲示板に操作性向上や機能拡張等を施しております。
●携帯電話
・拡張された機能等は利用できませんが閲覧／投稿は可能です。
・機種によっては正常に表示できない場合があります。
●スマートフォン(iPhone/Android)
・Safari(iPhone)
・Google Chrome
・Firefox
・Opera
・その他
●PC(Windows/Mac/Linux等)
・Microsoft Edge
・Internet Explorer 8以上
・Google Chrome
・Firefox
・Safari
・Opera 12.16 / 15以上
・その他
※緑色のブラウザは当フォーラムで推奨しているブラウザです。
※IE(Edge含む)は掲示板の仕様により十分な能力を引き出せない為です。
●テレビ／ゲーム機
・機種によっては表示不可や正常動作にならない場合があります。
